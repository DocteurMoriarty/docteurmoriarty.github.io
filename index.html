Base de documentation

Documentation technique simple en Markdown.

Les articles de la collection _articles/ sont publies automatiquement avec une mise en page sobre. Il suffit d'ajouter un fichier .md avec une categorie et une date.

Categories

Classement simple des sujets disponibles sur le site.

{% for category in site.data.categories %} {% assign posts_in_category = site.articles | where: "category", category.slug %}

{{ category.title }}

{{ category.description }}

{{ posts_in_category | size }} article{% if posts_in_category.size > 1 %}s{% endif %}
{% endfor %}

Derniers articles

Les derniers fichiers Markdown ajoutes dans _articles/.

{% assign sorted_articles = site.articles | sort: "date" | reverse %} {% for article in sorted_articles %} {% assign current_category = site.data.categories | where: "slug", article.category | first %}
{% if current_category %} {{ current_category.title }} {% endif %} {% if article.date %} {{ article.date | date: "%d/%m/%Y" }} {% endif %}

{{ article.title }}

{% if article.description %}

{% else %}

{{ article.excerpt | strip_html | truncate: 180 }}

{% endif %}
{% endfor %}
