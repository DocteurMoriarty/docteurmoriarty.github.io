Documentation technique

Ajoute tes articles en Markdown, le site s'occupe du rendu.

Chaque article de la collection _articles/ est publie automatiquement avec le meme style. Tu rediges en .md, tu choisis une categorie, et l'article apparait sur l'accueil.

Categories

Base prete pour organiser les sujets securite sans retoucher la structure HTML.

{% for category in site.data.categories %} {% assign posts_in_category = site.articles | where: "category", category.slug %}

{{ category.title }}

{{ category.description }}

{{ posts_in_category | size }} article{% if posts_in_category.size > 1 %}s{% endif %}
{% endfor %}

Derniers articles

Les fichiers Markdown places dans _articles/ sont listes ici automatiquement.

{% assign sorted_articles = site.articles | sort: "date" | reverse %} {% for article in sorted_articles %} {% assign current_category = site.data.categories | where: "slug", article.category | first %}
{% if current_category %} {{ current_category.title }} {% endif %} {% if article.date %} {{ article.date | date: "%d/%m/%Y" }} {% endif %}

{{ article.title }}

{% if article.description %}

{{ article.description }}

{% else %}

{{ article.excerpt | strip_html | truncate: 180 }}

{% endif %}
{% endfor %}
