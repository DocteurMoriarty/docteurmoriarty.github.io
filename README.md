# docteurmoriarty.github.io

Site GitHub Pages base sur Jekyll.

Les articles se mettent directement dans `_articles/` au format Markdown.

Exemple minimal :

```md
---
title: Mon article
description: Resume court
category: pentest
date: 2026-04-09
---

## Mon contenu

Texte en Markdown.
```
